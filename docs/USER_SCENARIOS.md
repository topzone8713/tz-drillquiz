# DrillQuiz 사용 시나리오 모음

이 문서는 DrillQuiz를 실제로 어떻게 활용할 수 있는지 다양한 시나리오를 담고 있습니다. 각 시나리오는 실제 사용자들이 겪을 수 있는 상황을 바탕으로 작성되었습니다.

---

## 📚 시나리오 1: 엄마와 아이의 숙제 복습 시간

**등장인물**: 엄마(김민지), 초등학교 3학년 아들(민수)

### 상황
민수가 오늘 학교에서 받아온 수학 숙제 파일이 있어요. 엄마는 이걸 활용해서 아이가 더 쉽게 이해할 수 있도록 연습 문제를 만들어주고 싶어합니다.

### 사용 과정

**1단계: 숙제 파일 업로드**
- 엄마가 DrillQuiz에 로그인하고 "Quiz Files" 메뉴로 들어가요
- 민수의 수학 숙제 파일(Excel이나 PDF)을 업로드해요
- 파일 제목을 "민수 수학 숙제 - 3학년 1학기"로 정하고 저장

**2단계: 아이 수준에 맞는 문제 만들기**
- "Exam Management"에서 새 시험을 만들기 시작해요
- 업로드한 숙제 파일에서 문제를 선택하는데, 엄마는 민수가 아직 어려워할 수 있으니 난이도를 "Easy"로 설정해요
- 문제 수는 10개로 정하고 "민수의 첫 번째 연습 시험"이라는 제목으로 저장

**3단계: 엄마와 아이의 스터디 그룹 만들기**
- "Study Management"에서 새 스터디를 만들어요
- 스터디 이름은 "민수와 엄마의 수학 공부방"으로 정하고, 목표는 "매일 10문제씩 꾸준히 풀기"로 설정해요
- 방금 만든 시험을 첫 번째 Task로 추가해요
- 민수를 멤버로 초대하려고 하는데, 민수는 아직 계정이 없으니 엄마가 아이 이름으로 간단히 추가해요

**4단계: 아이가 문제 풀기**
- 민수가 엄마 아이패드를 받아서 스터디에 들어가요
- "민수의 첫 번째 연습 시험"을 시작해요
- 문제를 하나씩 풀어가는데, 틀린 문제가 나오면 해설을 보고 이해해요
- 10문제를 다 풀고 결과를 확인해요 - 7문제 맞췄네요!

**5단계: AI 인터뷰로 실력 확인**
- 엄마는 민수가 정말 이해했는지 확인하고 싶어해요
- 시험 설정에서 "AI Interview Mode"를 켜요
- 민수가 다시 시험을 시작하는데, 이번엔 AI가 문제를 읽어주고 민수가 말로 답변해요
- AI가 민수의 답변을 듣고 정확한지 평가해주고, 틀렸다면 친절하게 설명해줘요

**6단계: 엄마에게 결과 전송**
- 시험이 끝나면 엄마는 결과를 이메일로 받고 싶어해요
- 결과 페이지에서 "Share with Parent" 버튼을 눌러요
- 엄마의 이메일 주소를 입력하고 전송해요
- 엄마는 이메일로 민수의 학습 결과를 받아서 확인해요:
  - 몇 문제를 맞췄는지
  - 어떤 부분을 어려워했는지
  - AI 인터뷰에서 어떻게 답변했는지
  - 다음에 집중해야 할 부분은 무엇인지

### 결과
엄마는 민수의 학습 상태를 한눈에 파악할 수 있고, 민수는 재미있게 공부하면서 실력을 키울 수 있어요. 매일 이렇게 반복하다 보면 자연스럽게 실력이 늘어나겠죠!

## 🎓 시나리오 2: 중학생 딸의 영어 단어 암기

**등장인물**: 엄마(박지영), 중학교 2학년 딸(서연)

### 상황
서연이가 영어 단어 시험을 앞두고 있는데, 단어장 파일을 받았어요. 엄마는 딸이 효율적으로 암기할 수 있도록 도와주고 싶어합니다.

### 사용 과정

**1단계: 단어장 파일 업로드**
- 엄마가 서연이의 영어 단어장 Excel 파일을 업로드해요
- 파일에는 단어, 뜻, 예문이 정리되어 있어요

**2단계: 단계별 시험 만들기**
- 엄마는 서연이 수준을 고려해서 세 가지 난이도로 시험을 만들어요:
  - "서연이 영어 단어 - 기초편" (Easy, 20문제)
  - "서연이 영어 단어 - 중급편" (Medium, 20문제)  
  - "서연이 영어 단어 - 고급편" (Hard, 20문제)

**3단계: 학습 스터디 구성**
- "서연이 영어 단어 암기 프로젝트"라는 스터디를 만들어요
- 세 개의 시험을 순서대로 Task로 추가해요
- 목표 기간을 2주로 설정해요

**4단계: 딸의 학습 진행**
- 서연이는 매일 조금씩 문제를 풀어요
- 틀린 단어는 자동으로 즐겨찾기에 추가되어 나중에 다시 볼 수 있어요
- 스터디 진행률을 보면서 자신의 성장을 확인해요

**5단계: AI 음성 인터뷰로 발음 연습**
- 서연이는 발음도 중요하다고 생각해요
- AI Interview Mode로 시험을 풀면서 단어를 직접 말로 발음해요
- AI가 발음을 평가하고 올바른 발음을 알려줘요

**6단계: 엄마에게 주간 리포트**
- 매주 금요일, 서연이의 학습 결과가 자동으로 엄마 이메일로 전송돼요
- 엄마는 딸의 진도와 정답률을 확인하고, 필요하면 격려 메시지를 보내요

### 결과
서연이는 체계적으로 영어 단어를 암기할 수 있고, 엄마는 딸의 학습 상태를 지속적으로 모니터링할 수 있어요. 2주 후 시험에서 좋은 성적을 받을 수 있을 거예요!

## 👨‍👩‍👧‍👦 시나리오 3: 형제가 함께하는 과학 퀴즈 대결

**등장인물**: 엄마(이수진), 초등 5학년 아들(준호), 초등 3학년 딸(예린)

### 상황
엄마는 두 아이가 함께 과학을 공부하면서 서로 경쟁하고 도와주는 분위기를 만들고 싶어해요.

### 사용 과정

**1단계: 과학 문제 파일 준비**
- 엄마가 두 아이 수준에 맞는 과학 문제 파일을 준비해요
- 초등학교 과학 교과서 내용을 바탕으로 문제를 만들었어요

**2단계: 각 아이 수준에 맞는 시험 만들기**
- 준호용: "준호의 과학 탐험 - 5학년" (Medium 난이도, 15문제)
- 예린용: "예린이의 과학 탐험 - 3학년" (Easy 난이도, 10문제)

**3단계: 가족 스터디 그룹 만들기**
- "우리 가족 과학 공부방"이라는 스터디를 만들어요
- 엄마, 준호, 예린 모두 멤버로 추가해요
- 각자의 시험을 Task로 추가하되, 서로의 시험도 볼 수 있게 공개 설정해요

**4단계: 형제의 학습 경쟁**
- 준호와 예린은 각자 자신의 시험을 풀어요
- 스터디 대시보드에서 서로의 진행률을 확인할 수 있어요
- "누가 더 빨리 끝낼까?" 하면서 재미있게 경쟁해요

**5단계: 서로 도와주기**
- 예린이 어려운 문제가 나오면, 준호가 옆에서 설명해줘요
- 준호도 예린이의 쉬운 문제를 보면서 기본기를 다시 확인해요

**6단계: 주말 AI 인터뷰 시간**
- 주말에는 가족이 함께 모여서 AI 인터뷰 모드로 시험을 풀어요
- 아이들이 직접 말로 답변하면서 표현력도 기를 수 있어요
- 엄마는 결과를 받아서 아이들의 이해도를 확인해요

### 결과
두 아이는 경쟁하면서도 서로 도와주는 좋은 학습 분위기를 만들 수 있어요. 엄마는 두 아이의 학습 상태를 한눈에 파악할 수 있고, 가족 모두가 함께 성장하는 기쁨을 느낄 수 있어요!

## 🏫 시나리오 4: 선생님과 학생들의 온라인 숙제 관리

**등장인물**: 선생님(최영희), 중학교 1학년 학생들(10명)

### 상황
선생님은 학생들의 숙제를 효율적으로 관리하고, 각 학생의 학습 상태를 파악하고 싶어해요.

### 사용 과정

**1단계: 숙제 문제 파일 업로드**
- 선생님이 이번 주 숙제 문제를 Excel 파일로 만들어서 업로드해요
- "1학년 1학기 3단원 복습 문제"라는 제목으로 저장해요

**2단계: 학생별 맞춤 시험 만들기**
- 선생님은 학생들의 수준을 고려해서 세 가지 난이도로 시험을 만들어요:
  - 기초반용 (Easy, 10문제)
  - 일반반용 (Medium, 15문제)
  - 심화반용 (Hard, 20문제)

**3단계: 학급 스터디 그룹 만들기**
- "1학년 1반 수학 숙제"라는 공개 스터디를 만들어요
- 학생들이 가입 요청을 보내면 선생님이 승인해요
- 각 학생의 수준에 맞는 시험을 Task로 추가해요

**4단계: 학생들의 숙제 수행**
- 학생들은 각자 계정으로 로그인해서 자신의 시험을 풀어요
- 문제를 풀면서 틀린 부분은 해설을 보고 이해해요
- 제출 기한(Due Date)이 설정되어 있어서 기한 내에 완료해야 해요

**5단계: 선생님의 진행 상황 확인**
- 선생님은 스터디 대시보드에서 각 학생의 진행률을 한눈에 확인해요
- "아직 안 푼 학생", "완료한 학생", "진행 중인 학생"을 구분해서 볼 수 있어요
- 각 학생의 정답률도 확인할 수 있어요

**6단계: 부진 학생 집중 관리**
- 정답률이 낮은 학생들을 선생님이 확인해요
- 해당 학생들에게 재시험을 만들어서 다시 풀게 해요
- AI 인터뷰 모드로 학생이 정말 이해했는지 확인할 수도 있어요

**7단계: 학부모 연락**
- 선생님은 필요시 학생의 학습 결과를 학부모 이메일로 공유할 수 있어요
- 학부모는 자녀의 학습 상태를 확인하고 집에서 추가로 도와줄 수 있어요

### 결과
선생님은 효율적으로 숙제를 관리할 수 있고, 학생들은 자신의 수준에 맞는 문제를 풀면서 실력을 키울 수 있어요. 학부모들도 자녀의 학습 상태를 투명하게 확인할 수 있어서 만족도가 높아요!

## 💼 시나리오 5: 직장인 부부의 자격증 시험 준비

**등장인물**: 부부(김대현, 이혜진), 둘 다 정보처리기사 자격증 준비 중

### 상황
부부가 함께 정보처리기사 자격증을 준비하고 있어요. 서로 경쟁하면서도 도와주는 분위기를 만들고 싶어해요.

### 사용 과정

**1단계: 기출 문제 파일 업로드**
- 대현이와 혜진이 모두 정보처리기사 기출 문제 파일을 업로드해요
- 각자 수집한 문제들을 통합해서 큰 문제 은행을 만들어요

**2단계: 과목별 시험 구성**
- 정보처리기사는 여러 과목이 있어서 과목별로 시험을 만들어요:
  - "정보처리기사 - 소프트웨어 설계" (20문제)
  - "정보처리기사 - 데이터베이스" (20문제)
  - "정보처리기사 - 운영체제" (20문제)
  - "정보처리기사 - 네트워크" (20문제)

**3단계: 부부 스터디 그룹**
- "대현&혜진 정보처리기사 합격 프로젝트"라는 스터디를 만들어요
- 두 사람 모두 멤버로 추가하고, 모든 시험을 Task로 추가해요
- 목표 기간을 시험일까지로 설정해요

**4단계: 각자의 학습 진행**
- 대현이와 혜진이는 각자 시간이 날 때마다 문제를 풀어요
- 틀린 문제는 자동으로 즐겨찾기에 추가되어 나중에 다시 볼 수 있어요
- 스터디 대시보드에서 서로의 진행률을 확인하면서 동기부여를 받아요

**5단계: 서로 도와주기**
- 한 사람이 틀린 문제가 나오면, 다른 한 사람이 설명해줄 수 있어요
- 스터디 내에서 코멘트 기능으로 서로 질문하고 답변해요

**6단계: 실전 연습 - AI 인터뷰**
- 시험 전에는 AI 인터뷰 모드로 실전 연습을 해요
- 면접 형식으로 문제를 받아서 말로 답변하는 연습을 해요
- AI가 답변을 평가하고 개선점을 알려줘요

**7단계: 최종 점검**
- 시험 전날, 부부는 각자의 학습 결과를 확인해요
- 부족한 부분을 파악하고 마지막으로 집중 복습해요
- 서로 격려하면서 시험에 임해요!

### 결과
부부는 함께 목표를 향해 나아가면서 서로를 격려하고 도와줄 수 있어요. 체계적인 학습 관리로 시험 합격 가능성이 높아지고, 부부 관계도 더 돈독해질 거예요!

## 🌍 시나리오 6: 해외 거주 가족의 한국어 학습

**등장인물**: 엄마(정미라), 해외 거주 중인 초등학생 딸(하은)

### 상황
해외에 거주하는 하은이는 한국어를 유지하기 위해 매일 공부해야 해요. 엄마는 딸의 한국어 학습을 체계적으로 관리하고 싶어해요.

### 사용 과정

**1단계: 한국어 교재 파일 준비**
- 엄마가 한국어 교재 내용을 Excel 파일로 정리해서 업로드해요
- 한글, 문법, 독해 문제들이 포함되어 있어요

**2단계: 단계별 학습 시험 만들기**
- 하은이의 한국어 수준에 맞춰서 단계별로 시험을 만들어요:
  - "하은이 한국어 - 기초 한글" (Easy, 15문제)
  - "하은이 한국어 - 기초 문법" (Medium, 15문제)
  - "하은이 한국어 - 독해 연습" (Medium, 10문제)

**3단계: 모국어 유지 스터디**
- "하은이 한국어 유지 프로젝트"라는 스터디를 만들어요
- 엄마와 하은이를 멤버로 추가하고, 모든 시험을 Task로 추가해요
- 매일 조금씩이라도 학습하는 것을 목표로 해요

**4단계: 딸의 일일 학습**
- 하은이는 매일 저녁 시간에 한국어 문제를 풀어요
- 문제를 풀면서 한국어 실력을 유지하고 향상시켜요
- 틀린 문제는 해설을 보고 이해해요

**5단계: AI 음성 인터뷰로 발음 연습**
- 하은이는 한국어 발음 연습도 중요해요
- AI Interview Mode로 문제를 풀면서 한국어로 직접 답변해요
- AI가 발음을 평가하고 올바른 한국어 표현을 알려줘요

**6단계: 엄마의 주간 모니터링**
- 매주 엄마는 하은이의 학습 결과를 이메일로 받아요
- 딸의 한국어 실력 향상을 확인하고, 필요하면 추가로 도와줄 수 있어요
- 해외에 있어도 딸의 학습 상태를 투명하게 확인할 수 있어요

**7단계: 다국어 지원 활용**
- DrillQuiz는 한국어와 영어를 모두 지원해요
- 하은이가 영어로 문제를 이해하고 싶을 때는 언어를 전환해서 볼 수 있어요
- 하지만 답변은 한국어로 해야 하니까 자연스럽게 한국어를 사용하게 돼요

### 결과
해외에 거주하는 하은이는 체계적으로 한국어를 유지할 수 있고, 엄마는 딸의 학습 상태를 지속적으로 모니터링할 수 있어요. 시간이 지나도 한국어 실력이 유지되고 향상될 거예요!
