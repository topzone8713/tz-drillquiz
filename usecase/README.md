# DrillQuiz Use Case 테스트 관리

DrillQuiz의 Use Case를 체계적으로 테스트하고 관리하기 위한 텍스트 기반 시스템입니다.

## 📁 폴더 구조
```
usecase/
├── README.md                              # 이 파일 (메인 가이드)
├── uc-{번호}-{타입}.md                   # 각 Use Case 테스트 보고서
└── scripts/                              # 자동화 테스트 스크립트
    ├── README.md                         # 스크립트 사용 가이드
    └── uc-{번호}-{타입}.sh               # 자동화 테스트 스크립트
```

## 🎯 시스템 특징
- ✅ **효율성**: 이미지 대신 텍스트로 빠른 로딩 및 검색
- ✅ **자동화**: API 테스트 스크립트로 반복 실행 가능
- ✅ **협업**: Git으로 변경사항 추적, 코드 리뷰 방식
- ✅ **확장성**: 새로운 Use Case 추가 용이
