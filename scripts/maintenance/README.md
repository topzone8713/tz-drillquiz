# Maintenance Scripts

시스템 유지보수 및 관리 작업을 위한 스크립트들입니다.

## 📁 Scripts

### `i18n-help.sh`
- **목적**: 국제화(i18n) 도구
- **사용법**: `./i18n-help.sh`
- **기능**: 
  - 번역 키 추출
  - 언어별 메시지 관리
  - 번역 파일 생성

### `translation-extractor.py`
- **목적**: 번역 키 자동 추출
- **사용법**: `python translation-extractor.py`
- **기능**: 
  - Vue 컴포넌트에서 번역 키 추출
  - Django 템플릿에서 번역 키 추출
  - 번역 파일 자동 업데이트

### `reset_password.py`
- **목적**: 사용자 비밀번호 재설정
- **사용법**: `python reset_password.py`
- **기능**: 
  - 특정 사용자 비밀번호 초기화
  - 개발/테스트 환경용
  - 기본 비밀번호로 재설정

## 🌐 국제화 지원

- **한국어 (ko)**: 기본 언어
- **영어 (en)**: 국제 사용자 지원
- **확장성**: 추가 언어 지원 가능

## ⚠️ 주의사항

- **백업**: 번역 파일 수정 전 백업
- **테스트**: 변경사항 테스트 필수
- **일관성**: 번역 키 네이밍 규칙 준수

## 🔄 작업 순서

1. 번역 키 추출
2. 번역 파일 업데이트
3. 번역 검증
4. 테스트 실행
5. 배포
